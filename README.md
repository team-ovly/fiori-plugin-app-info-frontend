# fiori-plugin-app-info-frontend
Fiori Plugin - App Info (frontend)

See also: [Backend repository](https://github.com/team-ovly/fiori-plugin-app-info-backend)

## How to contibute

Tool recomendations:
- VSCode
- Git
- NodeJS
- UI5 Tooling


Local installation:
- Clone this repository
- run `npm install`
- run `npm run start`
